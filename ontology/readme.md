Folder containing the core ontology, the constraint shapes (SHACL), and sample data.

We also added a copy of the ERA ontology, version 3.3.1, for demonstrating how POP interacts with external ontologies. For the latest ERA ontology releases, please consult [https://rinf.data.era.europa.eu/era-vocabulary/ontology.ttl].

OntoQA workflows read `ontology/src/config.md` (YAML front matter) to determine the ontology entrypoint (`main_ontology`) and the reasoner (`reasoner`).
